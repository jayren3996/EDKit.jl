# Model examples

This folder collects physics examples organized around specific models rather than a single API feature.

- `ConstrainedPXP.ipynb`: constrained Hilbert-space construction, diagonalization, and Neel-overlap diagnostics in the PXP model.
- `GapRatioXXZ.ipynb`: random-field XXZ chain and level-statistics analysis with `meangapratio`.
- `PXPDrive.ipynb`: driven PXP dynamics notebook.
- `Single-Particle.ipynb`: single-particle example notebook.
- `DiamonChain.ipynb`: diamond-chain notebook, with `diamonchain.jpg` as a companion figure.
- `InverseMethod.ipynb`: inverse-method notebook example.

These examples are a good second stop after the `Basic/` walkthroughs, once you want to see package pieces combined in concrete model studies.
