# Tensor-network examples

This folder collects examples centered on the ITensor-facing parts of `EDKit.jl`.

- `Tensors.ipynb`: broader tensor and ITensor workflow notebook.
- `MPSBasics.ipynb`: focused notebook on dense-to-MPS conversion, Pauli-space conversion, and bond-dimension inspection.

For the notebook-first introduction, start with `Basic/MPSAndPauli.ipynb`, then use this folder for the more tensor-network-focused material.
